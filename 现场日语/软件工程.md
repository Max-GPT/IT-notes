### プロシージャ
---
procedure
写在 数据库里 的一段程序

（ジョブ → バッチ → プロシージャ → PL/SQL）

可以被应用程序直接调用

用来做 数据处理、计算、批量更新

常见用途：

批量处理（バッチ処理）

复杂业务逻辑放在 DB 侧

提升性能 / 统一处理规则
- 夜間ジョブからバッチを起動し、
バッチ処理内（ない）でプロシージャを呼び出し、
データベースの集計処理を行う。
- データ更新用のプロシージャを作成する
- 作業プロシージャ
- この処理、プロシージャに切り出したほうがいいですね。
- 切り出す = 抽出来独立实现
- プロシージャが重いので、処理時間がかかっています。
- 本機能は、既存プロシージャを修正して対応します。
- ジョブ異常時は、該当プロシージャのログを確認してください。
- 夜間バッチジョブから、このプロシージャを呼び出します。
- データ更新処理はアプリではなく、プロシージャ側で実装しています。
- このプロシージャは、月次(げつじ)売上データを集計(しゅうけい)するためのものです。

| 项目   | プロシージャ | 関数（ファンクション） |
| ---- | ------ | ----------- |
| 返回值  | 不一定有   | 必须有         |
| 主要用途 | 执行处理流程 | 计算并返回结果     |
| 常见位置 | 数据库    | 程序代码 / DB   |
