## 快速生成构造函数
- 把光标放在类的花括号 {} 中
- 按下 `Alt` + `Shift` + `S`（打开 “Source” 菜单）
- 再按 `O`
- 勾选要生成构造方法的字段

## 快捷键
`Ctrl` + `Shift` + `F` 格式化文档 代码

`Alt` + `/` 提示代码

`Ctrl` + `+`  代码放大

`Ctrl` + `-`  代码缩小

`Ctrl` + `空格` 切换 Windows 中文 / 英语 输入法

## `.metadata` 
`.metadata` 文件夹是 工作空间 workspace 下自动生成的核心配置目录，非常重要。
