## 相关日语读法
- Eclipse エクリプス

## 快速生成构造函数
按下：

`Alt` + `Shift` + `S` → 再按 `O`

## Eclipse 中快速生成 Getter / Setter 
按下：

`Alt` + `Shift` + `S` → 再按 `R`

## 快捷键
`F3` 或者 `Ctrl` + `点击方法名` → 跳到定义

`Ctrl` + `Alt` + `H` → 查看调用层级（谁调用了它）

`Alt` + `←` / `→` → 页面前进后退

`Ctrl` + `Shift` + `F` 格式化文档 代码

`Alt` + `/` 提示代码

`Ctrl` + `+`  代码放大

`Ctrl` + `-`  代码缩小

`Ctrl` + `空格` 切换 Windows 中文 / 英语 输入法

## `.metadata` 
`.metadata` 文件夹是 工作空间 workspace 下自动生成的核心配置目录，非常重要。
